# Badge 标记
为按钮和文本提供提示标志。

## 基本用法
为按钮和文本提供提示标志.

<Playground>
  <Badge value={'20'}>
    <Button type="primary" size={'small'} onClick={() => {}}>icon</Button>
  </Badge>
</Playground>

## 自定义最大值
为该组件自定义最大值，如果当前值大于最大值，就返回最大值加+号

<Playground>
  <Badge value={20} max={19}>
    <Button type="primary" size={'small'} onClick={() => {}}>icon</Button>
  </Badge>
</Playground>

## 自定义标记类型
为该组件自定义标记类型，可设置为圆点或普通类型

<Playground>
  <Badge value={'20'}>
    <Button type="primary" size={'small'} onClick={() => {}}>icon</Button>
  </Badge>
  <div style={{'display': 'inline-block', 'width': '20px'}} />
  <Badge value={'20'} isDot={true}>
    <Button type="primary" size={'small'} onClick={() => {}}>icon</Button>
  </Badge>
</Playground>

## 属性

属性名|名称|类型|默认值
----|---|---|---
value|数值|[String, Number]|-
max|最大值|Number|-
isDot|是否为圆点|Boolean|false
